### 🚀 Избранные проекты

> **🤖 AI Telegram-бот для анализа БД**  
> *Превращаем естественный язык в SQL-запросы*  
> - **Решение:** Асинхронный бот (`aiogram`), который генерирует через LLM валидный SQL, выполняет его и возвращает ответ или ER-диаграмму.  
> - **Фишка:** Реализован механизм Self-Healing: при ошибке БД бот анализирует её, корректирует запрос и пробует снова.  
> - **Стек:** `Python`, `aiogram`, `SQLite`, `LLM API`.

> **📝 Test Prep Platform**  
> *Интерактивная платформа с алгоритмом интервального повторения*  
> - **Решение:** SPA на чистом `JavaScript` с рендерингом формул (`MathJax`). Неправильные ответы "прогоняются" заново до полного усвоения.  
> - **Результат:** **125+ студентов** используют регулярно. Работает по ссылке через **GitHub Pages** без бэкенда.  
> - **Стек:** `HTML5`, `CSS3`, `Vanilla JS`, `MathJax`.

> **📦 WB Automation Script**  
> *Инструмент для автоматизации сборки заказов Wildberries*  
> - **Решение:** Скрипт собирает данные о заказах в реальном времени, кластеризует похожие позиции и автоматически списывает их с локальной БД.  
> - **Стек:** `Python`, `SQLite`, `Requests`.
